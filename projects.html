Project 1: Game

Afew months ago, I used my Python programming knowledge and spent 3 weeks to program an entire game in the IDLE. The game played out like so: You are the Mayor of Simp City and your city consisted of a 4x4 board which you can build on every turn. There are five buildings and 8 copies of each. 2 buildings are selected randomly every turn and you select one of them to build. The game's objective was to score as high as possible. Each building scores differently depending on where they are built like for example, Beach is 3 points when on the right or left side of the map but only 1 point otherwise. The house gains a point for each different building adjacent to it but if there is a factory adjacent to it, it only scores 1 point no matter what.

Project 3: Study Application

I did coding using C# to create a desktop application that helps Students to study better. In the app, there is a tasklist, an option to select programs running in the desktop and set a certain amount of time to study. If students leave the app, they will be asked to return back to studying. If not they will be kicked from the app. There is also a chat function you can use to talk to teachers. You can link it to MS Teams in order to link the chats as well as make the tasklist automatically fill up whenever a teacher assigns a task.
